Requirements:

Necessary installation:
PyTorch 1.1.0+ (https://pytorch.org/get-started/locally/)
or 
TensorFlow 2.0+ (https://www.tensorflow.org/install)

Transformers (https://huggingface.co/docs/transformers/installation)

When installing transformers it will download the models:
DistilBERT base cased divided SQuAD (https://huggingface.co/distilbert-base-cased-distilled-squad)
pierreguillou/bert-base-cased-squad-v1.1-english (https://huggingface.co/pierreguillou/bert-base-cased-squad-v1.1-portuguese)
